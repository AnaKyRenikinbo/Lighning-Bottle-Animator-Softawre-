# Lighning-Bottle-Animator-Softawre-
A retro animatorsoftware I cobled together 
<h1>⚡ Lightning Bottle Animator</h1><p>A retro-styled animation and drawing application for Linux Mint, inspired by classic tools like Deluxe Paint and featuring a nostalgic 2000s web design aesthetic.</p><p><img src="https://img.shields.io/badge/Version-1.0.0-blue.svg" alt="Lightning Bottle Animator" class="e-rte-image e-imginline"> <img src="https://img.shields.io/badge/Platform-Linux%20Mint-green.svg" alt="Platform" class="e-rte-image e-imginline"> <img src="https://img.shields.io/badge/License-MIT-yellow.svg" alt="License" class="e-rte-image e-imginline"></p><h2>✨ Features</h2><h3>🎨 Drawing Tools</h3><ul> <li><strong>Brush Tool</strong> - Freehand painting with adjustable size</li> <li><strong>Airbrush</strong> - Smooth spray painting effect</li> <li><strong>Line Tool</strong> - Draw straight lines</li> <li><strong>Rectangle Tool</strong> - Filled and outline rectangles</li> <li><strong>Circle Tool</strong> - Filled and outline circles</li> <li><strong>Ellipse Tool</strong> - Filled and outline ellipses</li> <li><strong>Fill Tool</strong> - Flood fill enclosed areas</li> <li><strong>Eraser</strong> - Erase with adjustable size</li> <li><strong>Eyedropper</strong> - Pick colors from canvas</li> <li><strong>Selection Tool</strong> - Select areas and pick up brushes</li> <li><strong>Text Tool</strong> - Place text on canvas</li> </ul><h3>🎭 Character Templates</h3><p>Pre-built character templates inspired by classic animation books:</p><ul> <li>Comic Book Heroes</li> <li>Retro Cartoon Characters</li> <li>Cute Animals</li> <li>Robots &amp; Mechs</li> <li>Fantasy Characters</li> <li>Aliens &amp; Creatures</li> </ul><h3>📚 Layer System</h3><ul> <li>Multiple layers with visibility controls</li> <li>Layer reordering (move up/down)</li> <li>Per-layer editing</li> <li>Layer opacity support</li> </ul><h3>🎬 Animation Timeline</h3><ul> <li>Frame-by-frame animation</li> <li>Playback controls (play, pause, first, last, next, prev)</li> <li>Adjustable FPS (1-60)</li> <li>Frame duplication</li> <li>Thumbnail previews</li> </ul><h3>🎨 Color Palette</h3><ul> <li>32 preset colors</li> <li>Custom color picker</li> <li>Foreground/background color system</li> <li>Quick fill colors</li> </ul><h3>🖼️ File Support</h3><ul> <li>Open PNG, JPG, GIF, BMP images</li> <li>Save as PNG, JPG, GIF</li> <li>Export animations (GIF)</li> </ul><h2>🚀 Installation</h2><h3>Quick Install</h3><pre><code class="language-bash"># Extract the archive
